LifePlus
========

クックパッドの第4回 開発コンテスト24 に応募した作品です。

作品の説明
-------
プロポーズした、結婚した、子供が生まれたなど人生の節目節目の感動をツイッターでつぶやく亊で、数年後にその感動をタイムカプセルに入れてお届けします。
タイムカプセルを開くことで、あの頃の感動が蘇ります。

※コンテストで審査して頂くために、タイムカプセルが届く期間を数年後ではなく数分後としております。
